# 견적 단가 가이드

본 문서는 견적서 작성 시 적용하는 표준 단가와 계산 규칙을 정의한다.

## 표준 단가
- 클라우드 서버 구축의 표준 단가는 1대당 300만원이다 (부가세 별도).
- 온프레미스 LLM 도입 컨설팅의 표준 단가는 1건당 500만원이다 (부가세 별도).

## 공급가액 계산
- 공급가액은 단가 × 수량으로 계산한다.
- 예: 클라우드 서버 구축 단가 300만원, 수량 2대 → 공급가액 600만원.

## 부가세 정책
- 부가세는 공급가액의 10% 이다.
- 견적서에는 부가세를 포함하지 않고 "부가세 별도"로 표기하는 것이 원칙이다.
- 따라서 견적서의 '부가세 포함' 항목은 기본적으로 N(아니오)으로 표기한다.

## 비고
- 모든 견적서의 비고란에는 "부가세 별도"를 명시한다.
